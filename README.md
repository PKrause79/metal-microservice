# microservice
Microservice test platform

1. install Node.JS 8 LTS:

  https://nodejs.org/en/

2. install npm packages

  install.cmd

3. start all microservices

  start.cmd

4. start all microservices

  stop.cmd
 
5. tests

- static webpages -> browser

  http://127.0.0.1:5000

- api-gateway -> fiddler

  http://127.0.0.1:5001/api/demo
